# 第二週：熟悉 Visual Studio 2026 IDE

*Integrated Development Environment*

---

## 一、IDE 最基本操作 ✅

- 開啟舊有方案（Solution）
- 執行專案
- 結束執行
- 儲存方案中所有檔案

> **提示**：這些功能可從「檔案」或快速鍵 (Ctrl+Shift+O / F5 / Ctrl+Shift+S) 使用。

---

## 二、IDE 主要五個區塊 🧩

1. **Menu 命令列**

   - 初期只需熟悉常用的幾項選單（檔案、編輯、檢視、專案）。
2. **編輯區**

   - 顯示程式碼或設計畫面。
   - 每個 `*.cs` 檔案通常包含程式碼及對應的設計檢視。
3. **左側工具箱**

   - 可透過 `檢視 > 工具箱` 開啟。
   - 拖放常用控制項到設計畫面。
4. **右上角：方案/專案總管**

   - 管理方案與專案的檔案結構。
   - 可新增、刪除或重新命名檔案。
5. **右下角：屬性/事件窗格**

   - 顯示畫面上物件的屬性與可設定的事件。
   - 初期只掌握常用屬性即可。
   - 設計 UI 時需判斷何時使用 Label、TextBox、RadioButton、CheckBox、Button 等控制項。

---

## 三、練習/作業 1：建立 Hello World-3 方案 📝

建立一個包含以下控制項的 Windows Form：

| # | 控制項 | 屬性 | 值 |
|---|-------|-----|-----|
| 1 | Label | Text | "你支持台灣推動核能發電?" |
| 2 | RadioButton | Text | "支持" |
| 3 | RadioButton | Text | "不支持" |
| 4 | Button | Text | "確認送出" |

**按鈕事件邏輯**：

- **支持**：顯示訊息框
  ```csharp
  MessageBox.Show("贊成台灣推動核能發電-但需再三確保核電廠安全", "吳志成");
  ```

- **不支持**：顯示訊息框
  ```csharp
  MessageBox.Show("反對台灣推動核能發電-台灣不能有意外電貴很多也沒關係", "吳志成");
  ```
